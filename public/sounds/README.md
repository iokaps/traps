# Sound Effects

This directory contains sound effects for the Traps game.

## Required Sounds

Place the following MP3 files in this directory:

### Trap Sounds

- `ice-break.mp3` - Ice cracking/breaking sound (when player taps ice)
- `mud-clear.mp3` - Muddy splash/clear sound (when player swipes mud away)
- `trap-throw.mp3` - Whoosh/throw sound (when player throws a trap)

### Answer Sounds

- `correct.mp3` - Success chime/ding
- `incorrect.mp3` - Buzzer/wrong answer sound

### Timer Sounds

- `tick.mp3` - Clock tick for countdown
- `timer-warning.mp3` - Urgent warning sound (last 5 seconds)

### Game Phase Sounds

- `round-start.mp3` - Round beginning fanfare
- `round-end.mp3` - Round completion sound
- `game-over.mp3` - Final results fanfare

### UI Sounds

- `button-click.mp3` - Generic button click feedback
- `vote.mp3` - Vote submission confirmation

## Free Sound Resources

You can find free sound effects at:

- [Freesound.org](https://freesound.org)
- [Pixabay Sound Effects](https://pixabay.com/sound-effects/)
- [Zapsplat](https://www.zapsplat.com)

Make sure to check licenses before using any sounds.
